# AGENTS.md

## Project

PlantVision Web is a local web interface for the existing Python-based plant diagnosis workflow in this repository.

## Working Rules

- Read `.planning/PROJECT.md`, `.planning/REQUIREMENTS.md`, `.planning/ROADMAP.md`, and `.planning/STATE.md` before major implementation work.
- Keep Python as the diagnosis engine unless the user explicitly changes that decision.
- Prioritize the browser upload → diagnosis → full result flow over extra platform features.
- Preserve compatibility with the existing model behavior while refactoring for integration.
